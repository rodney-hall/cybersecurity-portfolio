### SOC Operations & Incident Triage

Built a Windows Server 2022 domain controller from scratch, promoted the server with AD DS and DNS roles, and designed a multi-region OU structure across USA, Europe, and Asia. Implemented departmental OUs with users, security groups, and computer objects organized for least-privilege access control at enterprise scale.

### Azure Cloud Security — RBAC & Least Privilege

Configured Azure Role-Based Access Control (RBAC) and Microsoft Entra ID, assigned scoped Reader roles to a test user, and confirmed least-privilege enforcement through documented access denial. Produced audit-ready evidence of the full IAM control chain from user creation through role assignment to denied write access.

### Network Traffic Analysis (Wireshark / PCAP)

Captured and analyzed OT network traffic using Wireshark. Applied protocol-specific display filters, decoded BACnet and Modbus/TCP communication patterns, and correlated request-response sequences to establish behavioral baselines. Identified unauthenticated command traffic and device enumeration behavior relevant to threat detection and detection rule development.

## OT/ICS Security — Specialization

### OT Network Threat Detection & Incident Analysis

Detected and analyzed suspicious control activity in OT network traffic using Wireshark. Identified unauthorized Modbus function codes with potential to alter device state, flagged BACnet broadcast traffic exhibiting active reconnaissance patterns, and produced findings and recommendations aligned to ICS incident response procedures.

**[View Project →](https://rodney-hall.github.io/cybersecurity-portfolio/projects/ot-network-threat-detection/)**

### OT/ICS Foundations — Purdue Model & IT/OT Segmentation

Designed IT/OT network segmentation architecture using the Purdue Model (Levels 0–5), documented an OT DMZ topology to isolate operational systems from enterprise IT, and produced an incident evidence checklist to support analyst triage and escalation in OT environments.

**[View Project →](https://rodney-hall.github.io/cybersecurity-portfolio/projects/ot-ics-foundations/)**

Enumerated running services, listening network ports, and active interfaces. Documented findings to support change detection and safe escalation decisions without disrupting operations.
